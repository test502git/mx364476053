企业版用户
让企业拥有一套私有FOFA
以往，企业客户仅能通过网站查询、网络服务等形式进行网络空间资产搜索操作。
而FOFA能够以私有化形式部署于企业内部，不仅有更高的资产检索效率，同时兼具更低的使用成本。
使用 FOFA 的优势
AVANTAGES
HTML代码级别全文检索
可以针对包括标签在内的HTML代码级全文索引和检索，相比同类产品而言其检索内容更丰富，效果更快速、更精准，可大幅提高用户的工作效率；
预置广泛的特征规则
产品预置超过800条特征规则，覆盖软硬件，安全设备，邮件系统，OA业务系统等，且用户可以随时自行动态增加，灵活度更高；
深厚的全网数据积累
拥有行业领先的超过1.46亿的网络空间资产基因库，与高效的数据检索技术，能够快速提高用户的网络资产发现能力；
高可靠性、高成熟度
目前FOFA产品已经在实际应用场景中稳定运行超过2年，并成功部署于各类企业用户的生产环境中，多次发挥重要作用；
FOFA 的功能及特点
更多精彩为企业护航
  • 更全面的资产发现
    支持主动扫描、流量监控、搜索引擎、社交平台等多种资产采集方式；
  • 多种协议分析
    提供包括HTTP协议、数据库协议、工控协议在内的多种协议分析功能；
  • 详细字段分析
    支持针对每种协议进行详细数据字段的分析，如包括SSL证书细节内容等；
  • 标签化资产特征提取
    对采集到的资产进行标签化的特征提取，包括软硬件、操作系统、应用产品等，便于分类统计和快速检索；
  • 支持动态查询语句
    支持高级查询语句与动态查询语句，实时、高效检索大数据内容；
  • 分布式平台
    采用分布式数据采集和检索平台，可根据规模大小进行动态平滑扩展；
  • 多维度统计报表
    提供包括地理位置统计、端口分布、协议分布等多维度统计报表功能；
  • 私有化部署方式
    提供私有化部署方式，可对内网资产进行扫描和检索，搭建企业自有检索平台；
  • API 查询接口
    支持API查询接口，便于进行程序间的自动化查询、处理功能，进行进一步的业务分析；
  • 全网漏洞扫描引擎
    内置全网漏洞扫描引擎，用户可编写私有的漏洞规则，针对漏洞规则进行分钟级别的快速验证；
